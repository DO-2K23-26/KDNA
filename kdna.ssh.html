connect()[source]

Connects to the specified host using the provided user credentials.

If a connection has not been established yet, this method creates a new connection using the given host and user credentials. The connection is established with the help of the Connection class, which is initialized with the host, user, and connection kwargs obtained from the _get_connect_kwargs method.

Returns:

The established connection.

Return type:

Connection
